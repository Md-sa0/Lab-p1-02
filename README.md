Laboratório P1: Transformer Encoder "From Scratch"Este repositório contém a minha implementação da passagem direta (Forward Pass) de um bloco Encoder do Transformer, baseado no artigo "Attention Is All You Need". O código foi feito do zero, cumprindo a restrição de usar apenas numpy e pandas, sem bibliotecas de Deep Learning prontas como PyTorch ou TensorFlow.COMO RODAR O CÓDIGOCertifica-te de que tens o Python 3.x instalado.Instala as dependências necessárias executando no terminal:pip install numpy pandasExecuta o script principal no teu terminal:python transformer_encoder.pySAÍDA ESPERADA (SANITY CHECK)Ao executar o código, o tensor de entrada (que representa a nossa frase mapeada) vai atravessar sequencialmente as N=6 camadas do Encoder.Conforme exigido pelo roteiro da prática, o tensor deve manter as suas dimensões inalteradas entre a entrada e a saída, provando que as multiplicações de matrizes, projeções e conexões residuais estão corretas. O terminal deverá imprimir exatamente o seguinte:Entrada (Batch, Tokens, Dim): (1, 5, 64)Saida (Batch, Tokens, Dim): (1, 5, 64)Nota: Estamos a usar d_model = 64 em vez de 512 (como no artigo original) para facilitar o processamento local na CPU, conforme autorizado pelo professor.
